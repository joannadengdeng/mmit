手算 Transformer Block -- RMSNorm + Attention + SwiGLU

用 seq_len=2, dim=4, n_heads=1, mlp_dim=6 手算 LLaMA 风格的一个 Transformer Block。

双栏排版，打印 A4 纵向。Part 0 数据页可撕下放旁边，答案在最后一页。

整个 Block 的数据流：
x --> RMSNorm --> Causal Attention --> + residual --> RMSNorm --> SwiGLU MLP --> + residual --> LM Head --> loss

操作公式

SoftmaxP(i) = exp(x_i) / sum_j exp(x_j)
Cross-Entropyloss = -log(P(correct))
